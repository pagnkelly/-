## 这是什么

  实现web聊天的@功能，详细如下

  - @后出成员列表
  - 按一下退格键可删除全部成员名字
  - 支持点击和键盘上下键操作
  - 艾特列表动态跟随
  
## 例子

 - clone工程打开dist中的index.html

 - 启动工程

 ```shell
    npm install | yarn install
    npm start
 ```
