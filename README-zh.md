## 这是什么

  实现web聊天的@功能，详细如下

  - @后出成员列表
  - 按一下退格键可删除全部成员名字
  - 支持点击和键盘上下键操作
  - 艾特列表动态跟随

## 怎样实现

之前用的是低版本的```react```，这次用了新API(```hooks```)升级了一下，除此之外，富文本编辑器用的```kindeditor```，还有```jquery```，包括```caret```用来解决跟随光标位置问题。

## 例子

 - clone工程打开dist中的index.html

 - 启动工程

 ```shell
    npm install | yarn install
    npm start
 ```
